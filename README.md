# المركز الفني للسيارات

This is a Next.js application for "المركز الفني للسيارات", an auto repair workshop specializing in German, European, and Chinese cars. This project was built using Next.js, React, Tailwind CSS, and ShadCN UI, and prototyped in Firebase Studio.

## Getting Started

First, install the project dependencies:
```bash
npm install
```

Next, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the application.

## About This Project

This website serves as the online presence for the auto workshop. It features:

- A showcase of specialized services.
- A list of supported car brands.
- A gallery for bodywork repairs.
- Contact information and business hours.
- AI-generated content for service descriptions via Genkit.
- A responsive design for all screen sizes.
