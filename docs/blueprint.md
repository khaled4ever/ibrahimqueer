# **App Name**: AutoKraft Zentrum

## Core Features:

- Service Catalog: Showcase the workshop's specialties: mechanics, engine/gearbox repairs, electrical diagnostics, auto body & paint, and routine maintenance.
- Specialty Filters: Allow users to filter services specifically for German and European cars.
- AI-Powered Image Generation: Use a tool to generate appropriate visuals for each service section. For example, use generative AI to create an image of an engine under repair.
- Auto Body Slider: Include an image slider or carousel for the Auto Body & Paint section to display before and after shots.
- Brand Recognition Slider: Display a banner image, then beneath it a slider featuring the logos of German automotive brands.
- WhatsApp Chat Button: Implement a floating WhatsApp button with a 'Message Us' label.
- Click-to-Call Button: Include a floating call button labeled 'Call Us'.

## Style Guidelines:

- Primary color: Deep, rich blue (#1E3A8A) to convey trust, expertise, and reliability, aligning with the precision associated with German automotive engineering.
- Background color: Very light, desaturated blue (#F0F4FF), providing a clean, professional backdrop that doesn't distract from the content.
- Accent color: Muted purple (#7E57C2) to highlight important CTAs and sections.
- Body font: 'Inter', a sans-serif for a modern, neutral, machined feel. Use for body text.
- Headline font: 'Space Grotesk', a sans-serif with a computerized, techy, scientific feel. Use for headlines.
- Use icons that are simple, geometric, and related to automotive maintenance and repair, consistent with a modern aesthetic.
- Ensure a clean, structured layout with clear section divisions to showcase each service area distinctly.
- Use subtle transitions and loading animations to provide a smooth and polished user experience.
